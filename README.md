# Vulkan Craft

## Resources

### Graphics
 - [**Vulkan Documentation**](https://docs.vulkan.org/spec/latest/)
 - [**Vulkan Tutorial**](https://vulkan-tutorial.com/): The best reference for getting started with Vulkan. One should also look at an updated adaptation of this one [here](https://docs.vulkan.org/tutorial/latest/00_Introduction.html) published directly by Khronos.
 - [**Vulkan Guide**](https://vkguide.dev/): Provides good reference for how to structure an Vulkan-backed engine using some modern Vulkan features.
 - [**Vulkan Samples**](https://github.com/SaschaWillems/Vulkan): Best samples for implementing several Vulkan techniques.
### Networking
 - [**Getting Started with Winsock**](https://learn.microsoft.com/en-us/windows/win32/winsock/getting-started-with-winsock)
 - [**Building a Game Network Protocol**](https://gafferongames.com/categories/building-a-game-network-protocol/): Gives good advice for building networks over UDP, including information on bitpacking, packet fragmentation, and reliability. Hightly recommended!
 - [**Making a Multiplayer FPS**](https://codersblock.org/multiplayer-fps/): Some useful implementation details on handling multiple clients and client-side prediction.
